### 2. Project Rules 設計

| 項目      | 内容                                                                                                                                             |
| ------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| **要件**  | Cursor Agent と Cmd-K 補完の両方に適用される包括ルールを作成 ([Cursor][1])                                                                                         |
| **手順**  | 1. `.cursor/rules` を作成（後述の雛形を貼付）<br>2. 主要ディレクトリごとにセクション分割：`agents/*`, `tools/*`, `templates/*`<br>3. `@file:` 記法でサンプルテンプレを明示し、Agent にレファレンスさせる |
| **検証**  | Cursor で Cmd-K → “Explain file” を実行し、ルール反映を確認                                                                                                  |
| **成果物** | `.cursor/rules` 完成版（Git 管理対象）                                                                                                                  |
